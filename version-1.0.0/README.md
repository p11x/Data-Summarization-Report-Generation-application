# Version 1.0.0

This folder contains the initial release of the Data Summarization Report Generation Application.

## Features

- **Authentication**: Login and signup functionality with route protection
- **Dashboard**: Main dashboard with navigation to all features
- **Profile**: User profile page with editable details
- **Settings**: Application settings (appearance, notifications, privacy)
- **Topics**: Browse topics from CSV data with single-click navigation
- **Topic Detail**: Full report-like analysis for each topic including:
  - Input Data Overview
  - Automated Summary with insights
  - Data Visualizations (bar, pie, doughnut, line charts)
  - Key Metrics
  - Detailed Data Table with sorting/filtering/pagination
  - Column Analysis
  - Conclusion with recommendations
  - Export functionality (CSV and text)
- **Upload**: CSV file upload functionality
- **Report**: Data analysis report generation
- **Download**: Download generated reports

## Technology Stack

- Angular 19
- TypeScript
- Chart.js (ng2-charts)
- CSS3

## Getting Started

1. Install dependencies: `npm install`
2. Start the application: `npm start`
3. Navigate to `http://localhost:4200`
