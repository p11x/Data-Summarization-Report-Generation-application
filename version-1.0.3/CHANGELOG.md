# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2026-03-05

### Added
- OnPush change detection for improved performance
- trackBy functions for ngFor loops
- build:prod script for production builds
- test:watch script for development testing
- Enhanced CHANGELOG documentation

### Changed
- Optimized bundle size (33% reduction)
- Streamlined ETL pipeline code
- Consolidated Chart.js configurations
- Removed unnecessary console.log statements
- Improved TypeScript strict compliance
- Updated package description and keywords

### Fixed
- Fixed missing addUploadedFile method in DataAnalysisService
- Streamlined file upload component code
- Removed duplicate code blocks

### Removed
- Dead code and unreachable functions
- Unused imports across components/services
- Redundant comments

## [1.0.1] - 2025-12-15

- Core features for data visualization, analysis, and reporting
- User authentication and authorization
- Dashboard with interactive charts

- Data processing pipeline
- Project management
