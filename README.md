# Data Summarization Report Generation Application

A web application for data analysis and report generation built with Angular.

## Project Structure

```
├── version-1.0.0/          # Initial release (stable version)
│   ├── src/                # Source code
│   ├── plans/              # Architecture documentation
│   ├── README.md           # Version 1.0.0 documentation
│   └── ...config files
│
├── version-1.0.1/          # Future changes and additions
│   └── README.md           # Version 1.0.1 documentation
│
├── src/                    # Working source code (for development)
├── plans/                  # Working documentation
└── ...config files         # Angular configuration
```

## Features

- **Authentication**: Login and signup functionality with route protection
- **Dashboard**: Main dashboard with navigation to all features
- **Profile**: User profile page with editable details
- **Settings**: Application settings (appearance, notifications, privacy)
- **Topics**: Browse topics from CSV data with single-click navigation
- **Topic Detail**: Full report-like analysis for each topic
- **Upload**: CSV file upload functionality
- **Report**: Data analysis report generation
- **Download**: Download generated reports

## Getting Started

1. Install dependencies: `npm install`
2. Start the application: `npm start`
3. Navigate to `http://localhost:4200`
